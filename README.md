Это репозиторий с инфраструктурой для учебного проекта курса DevOps Basic.
В разделе terraform лежат скрипты и роли для разворачиния в облаке инфраструктуры из лоадбалансера, и набора виртуалок для серверов приложений, гитлаба, мониторинга, системы упраления логами.
Вся инфраструктура мной разворачивалась в Google Cloud.
В разделе ansible на этих ВМ разворачиваются собственно сервера приложений с Docker, так как приложение стартует из контейнера, gitlab-ce, prometheus, grafana, loki. На графану и прометеус еще настроены 
реверс-прокси nginx, чтобы пробрасывать их рабочие порты на 80 наружу. Ну и плюс агенты для прометеуса и локи.

Само приложение и автоматизация CI/CD в соседнем репозитории service.
